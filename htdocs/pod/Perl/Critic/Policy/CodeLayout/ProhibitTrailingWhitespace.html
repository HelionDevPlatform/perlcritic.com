NAME

Perl::Critic::Policy::CodeLayout::ProhibitTrailingWhitespace - Don't use whitespace at the end of lines.

AFFILIATION

This Policy is part of the core Perl::Critic distribution.

DESCRIPTION

Anything that is not readily visually detectable is a bad thing in general, and more specifically, as different people edit the same code, their editors may automatically strip out trailing whitespace, causing spurious differences between different versions of the same file (i.e. code in a source control system).

CONFIGURATION

This Policy is not configurable except for the standard options.
