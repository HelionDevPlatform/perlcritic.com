NAME

Perl::Critic::Policy::Modules::RequireNoMatchVarsWithUseEnglish

DESCRIPTION

Due to unfortunate history, if you use the English module but don't pass in a -no_match_vars argument, all regular expressions in the entire program, not merely the module in question, suffer a significant performance penalty. See the English documentation for details. 

  use English;                              # not ok
  use English '-no_match_vars';             # ok
  use English qw< $ERRNO -no_match_vars>;   # ok
