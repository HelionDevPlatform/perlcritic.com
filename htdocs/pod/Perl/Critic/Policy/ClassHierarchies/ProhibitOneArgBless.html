NAME

Perl::Critic::Policy::ClassHierarchies::ProhibitOneArgBless

DESCRIPTION

Always use the two-argument form of bless because it allows subclasses to inherit your constructor. 

  sub new {
      my $class = shift;
      my $self = bless {};          # not ok
      my $self = bless {}, $class;  # ok
      return $self;
  }
