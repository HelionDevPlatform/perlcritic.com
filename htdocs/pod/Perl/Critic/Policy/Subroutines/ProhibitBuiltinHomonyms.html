NAME

Perl::Critic::Policy::Subroutines::ProhibitBuiltinHomonyms

DESCRIPTION

Common sense dictates that you shouldn't declare subroutines with the same name as one of Perl's built-in functions. See perldoc perlfunc for a list of built-ins. 

  sub open {}  #not ok
  sub exit {}  #not ok
  sub print {} #not ok

  #You get the idea...

Exceptions are made for BEGIN, END, INIT and CHECK blocks, as well as AUTOLOAD and import subroutines.
