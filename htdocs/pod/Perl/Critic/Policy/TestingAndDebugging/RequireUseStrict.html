NAME

Perl::Critic::Policy::TestingAndDebugging::RequireUseStrict - Always use strict.

AFFILIATION

This Policy is part of the core Perl::Critic distribution.

DESCRIPTION

Using strictures is probably the single most effective way to improve the quality of your code. This policy requires that the 'use strict' statement must come before any other statements except package, require, and other use statements. Thus, all the code in the entire package will be affected.

There are special exemptions for Moose and Moose::Role because they enforces strictness; e.g. 'use Moose' is treated as equivalent to 'use strict'.

The maximum number of violations per document for this policy defaults to 1.

CONFIGURATION

This Policy is not configurable except for the standard options.
