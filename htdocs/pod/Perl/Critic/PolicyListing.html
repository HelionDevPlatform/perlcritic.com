NAME

Perl::Critic::PolicyListing - Display information about Policies

DESCRIPTION

This is a helper class that formats a set of Policy objects for pretty-printing. There are no user-serviceable parts here.

CONSTRUCTOR

new( -policies => \@POLICY_OBJECTS )
Returns a reference to a new Perl::Critic::PolicyListing object.

METHODS

short_listing()
Returns a list of strings, where each string is a one-line summary of the default severity, policy name, and default themes for each Policy that was given to the constructor of this PolicyListing.
long_listing()
Returns a list of strings, where each string is a multi-line summary of the policy name, default themes, and default severity for each Policy that was given to the constructor of this PolicyListing. The format is suitable for use in the .perlcriticrc file.
