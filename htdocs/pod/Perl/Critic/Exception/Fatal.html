NAME

Perl::Critic::Exception::Fatal - A problem that should cause Perl::Critic to stop running.

DESCRIPTION

Something went wrong and processing should not continue. You should never specifically look for this exception or one of its subclasses.

Note: the constructor invokes Exception::Class/"show_trace" to force stack-traces to be included in the standard stringification.

This is an abstract class. It should never be instantiated.

METHODS

full_message()
Overrides Exception::Class/"full_message" to include extra information.
